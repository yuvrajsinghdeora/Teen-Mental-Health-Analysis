# 🧠 Teen Mental Health Analysis Using Python

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on a Teen Mental Health dataset using Python.

## 🎯 Project Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a Teen Mental Health dataset to identify patterns and relationships between social media usage, sleep habits, stress levels, anxiety, addiction, and academic performance using Python.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Visualizations
- Gender Distribution
- Platform Usage Distribution
- Sleep Hours Histogram
- Correlation Heatmap
- Academic Performance vs Addiction (Regression Plot)

## 📁 Dataset
Teen_Mental_Health_Dataset.csv

## 📄 Report
Complete project report included in this repository.

## 🔍 Key Insights

- Social media usage varies across different platforms.
- Sleep duration shows variation among teenagers.
- Correlation analysis indicates weak relationships between most numerical variables.
- Regression analysis suggests a weak relationship between addiction level and academic performance.
- Visualizations help understand the dataset more effectively.
